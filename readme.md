## Instalação 
### Requisitos
 - Docker
 - Porta 8080 liberada.
 - Porta 3000 liberada.

Execute um dos seguintes comandos:

bash
```
bash install.sh
```
ou
CMD
```
install
```
na pasta raiz do projeto

## Iniciando Loteria 
Execute um dos seguintes comandos:
bash
```
bash run.sh
```
ou
CMD
```
run
```
na pasta raiz do projeto

Endereço de acesso: http://localhost:3000
